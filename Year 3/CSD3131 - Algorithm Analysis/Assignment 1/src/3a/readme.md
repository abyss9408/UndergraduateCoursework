How to build and run the program:
    1. ensure the computer has make
    2. run the command inside the folder: make run SIZE={size}
        2a. {size} = 10k (default), 100k, 1M
        2b. example: make run SIZE=1M
    3. the build program will be stored as 3a_sorting
    4. the program will export 2 files, counting_sort.txt, selection_sort.txt
